# Optional StyleStack help images

The help screen already works with polished illustrated fallbacks. Replace those
fallbacks with real, landscape WebP/JPEG screenshots by adding these files:

- `item_photo.webp` — a full garment, fully visible, in bright even light.
- `todays_outfit.webp` — the Today outfit canvas and its explanation.
- `outfit_log.webp` — the logging action and the resulting outfit history.
- `style_tools.webp` — Canvas Style Builder with a few arranged items.

Recommended size: 1200 x 750 px. Avoid personal data in screenshots.
